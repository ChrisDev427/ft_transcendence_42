ft_transcendence

Welcome to ft_transcendence

The last common core project at school 42

The ft_transcendence project is a complex and ambitious endeavor aiming to create an online gaming platform based on table tennis (ping-pong). It is a large-scale project implementing diverse features, ranging from user management to real-time table tennis matches. It is a versatile project that combines aspects of web development, real-time functionality, and tournament management to craft an immersive and competitive online gaming experience centered around table tennis. It challenges students' skills in software development, database design, security, and the integration of modern technologies.
