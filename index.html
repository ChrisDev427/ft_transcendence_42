Welcome !

This is the last common core project at school 42.
It implements a multiplayer platform for the game of Pong.
The project aims to develop skills in web programming, database, security and communication between the frontend and backend.
Players can register, create tournaments, and play Pong against other users.
The goal is to create a robust and secure real-time gaming experience.
